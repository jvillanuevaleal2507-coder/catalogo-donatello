# Catálogo Ventas Donatello

Catálogo público conectado a Supabase.

## Variables de entorno en Vercel

Agrega las mismas variables del POS:

- VITE_SUPABASE_URL
- VITE_SUPABASE_ANON_KEY

## WhatsApp

En `src/App.jsx`, cambia:

```js
const WHATSAPP_NUMBER = "528999999999";
```

por el número real de Ventas Donatello en formato internacional.
